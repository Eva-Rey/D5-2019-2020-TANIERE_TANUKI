    Règles du jeu

    Les joueurs se placent en face à face les 2 index à l’horizontal. Les doigts horizontaux correspondant aux doigts en jeu. Les joueurs ont donc chacun un doigt à chaque main. n joue chacun son tour dans le sens des aiguilles d’une montre.

    À chaque tour, un joueur utilisera l’une de ses mains pour toucher l'une des mains de son adversaire. Si vous tapez avec un doigt, votre adversaire ajoutera votre(s) doigt(s) + ses doigts déjà levés. Votre adversaire ajoute alors sur sa main tapée le nombre de doigts qui lui a été pointé.

    À tour de rôle, les joueurs se tapent la main. Le but est de continuer et d'ajouter des doigts à la main de votre adversaire en tapotant. Lorsque la main de quelqu'un atteint cinq doigts levés, cette main est éliminée du jeu. Continuez à jouer jusqu'à ce qu'un joueur ait perdu ses deux mains. Le but est d'être le dernier à rester avec au moins une main encore vivante.

    Rendez le jeu plus intéressant en ajoutant une nouvelle règle. Introduisez les scissions dans le jeu. Quand c'est votre tour, vous pouvez taper vos deux mains ensemble pour redistribuer le nombre de doigts que vous avez étendus. Par exemple, si vous avez une main avec trois doigts et une main avec un seul doigt et que vous les divisez, vous pouvez placer deux doigts sur chaque main. Si vous avez une combinaison telle que quatre doigts et un doigt sur chaque main, vous pouvez la diviser en une main de trois et une main de deux. L'objectif de cette stratégie est d'empêcher l'une de vos mains d'atteindre cinq doigts et d’être éliminé.

    Un joueur peut "faire revivre" une main morte en se dédoublant. Si vous avez une main morte et une vivante avec quatre doigts, vous pouvez séparer et mettre deux doigts sur chaque main pour ramener votre main morte dans le jeu.
